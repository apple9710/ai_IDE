# Claude IDE

Claude Code를 실행하기 위한 가벼운 VS Code 스타일 IDE. (Electron 기반)

> 📖 자세한 문서는 [`docs/`](./docs/) 참고 — [기능 가이드](./docs/features.md) · [아키텍처](./docs/architecture.md)

## 레이아웃

```
┌──┬────────────┬─────────────────────┬──────────┐
│활│ 폴더 트리   │   코드 에디터        │  Claude  │
│동├────────────┴─────────────────────┤  채팅    │
│바│          터미널                   │  패널    │
└──┴───────────────────────────────────┴──────────┘
```

- **가장 좌측 (활동 바)**: `📁` 탐색기 토글 / `💻` 터미널 토글 / `💬` Claude 채팅 토글
- **사이드바**: 폴더 트리 (클릭으로 폴더 펼치기, 파일 클릭으로 에디터에 열기)
- **중앙**: Monaco 코드 에디터 (탭 지원, `Ctrl+S` 저장)
- **하단**: 실제 PTY 터미널 (`@lydell/node-pty` + xterm.js) — 일반 셸
- **우측**: Claude 채팅 패널 (Agent SDK)

## 실행

```bash
npm install      # 최초 1회
npm start        # 실행
```

또는 `start.bat` 더블클릭.

## 주요 기능

- **Claude 채팅 패널** — 우측 패널에서 채팅형 UI로 Claude Code 사용 (기본 방식, Agent SDK)
- 하단 터미널은 일반 셸 (npm/git 등 자유롭게 사용, `＋`로 여러 개)
- 마지막에 연 폴더 자동 복원
- 하단 터미널 / 우측 채팅 패널 크기 드래그 조절
- **Git 상태 표시** — 브랜치명 + 파일별 상태 배지 (M 수정 / A 추가 / D 삭제 / U 미추적), 변경 포함 폴더에 점 표시
- **파일/폴더 생성·삭제·이름변경** — 사이드바 헤더 버튼 또는 트리 우클릭 컨텍스트 메뉴
- **Claude 데스크톱 스타일 디자인** — 웜 다크 테마 + 클레이(coral) 액센트
- **Claude 채팅 패널** — 우측 패널에서 채팅형 UI로 Claude Code 사용 (Agent SDK 기반)

## Claude 채팅 패널 (Agent SDK)

우측 `✳ Claude` 패널은 `@anthropic-ai/claude-agent-sdk`로 Claude Code 엔진을 구동합니다.
터미널이 아닌 **채팅형 UI**로, 데스크톱 앱처럼 사용할 수 있습니다.

- 스트리밍 응답(토큰 단위), 툴 호출 카드(펼치면 입력/결과 확인), thinking 표시
- 파일 편집·명령 실행 등 권한이 필요한 작업은 **허용 / 항상 허용 / 거부** 카드로 승인
  - 이미 허용 규칙이 있는 작업은 자동 허용됩니다 (CLI와 동일)
- 대화는 열려 있는 폴더(`cwd`)를 기준으로 동작
- `＋ 새 대화`로 세션 초기화, `■`로 진행 중 작업 중단
- 단축키: `Ctrl+L` (채팅 패널 토글/포커스)

> 채팅 패널은 터미널의 `claude`와 별개로 동작하며, 동일한 Claude Code 로그인/인증을 사용합니다.
> 터미널에서 `claude` 로그인이 되어 있으면 별도 설정 없이 바로 사용 가능합니다.

## Git 상태 배지

| 배지 | 의미 | 색상 |
|------|------|------|
| `M` | 수정됨 | 앰버 |
| `A` | 스테이징된 추가 | 그린 |
| `D` | 삭제됨 | 레드 |
| `U` | 미추적 파일 | 블루 |
| `R` | 이름 변경 | 블루 |
| `●` | (폴더) 내부에 변경 존재 | 앰버 |

> Git 저장소가 아닌 폴더를 열면 브랜치 바는 자동으로 숨겨집니다.
> 파일 저장·생성·삭제 후 상태는 자동 새로고침되며, 🔄 버튼으로 수동 새로고침도 가능합니다.

## 단축키

| 키 | 동작 |
|----|------|
| `Ctrl+B` | 탐색기 토글 |
| `` Ctrl+` `` | 터미널 토글 |
| `Ctrl+L` | Claude 채팅 패널 토글/포커스 |
| `Ctrl+S` | 현재 파일 저장 |

## 인증

Claude 채팅 패널은 설치된 **Claude Code CLI 로그인(`claude`)을 그대로 사용**합니다.
터미널에서 한 번 `claude` 로그인을 해두면 채팅 패널도 별도 설정 없이 동작합니다.

## 기술 스택

- **Electron 33** — 데스크톱 셸
- **@anthropic-ai/claude-agent-sdk** — Claude 채팅 엔진 (ESM, main에서 dynamic import)
- **@lydell/node-pty** — 프리빌드 PTY (컴파일 불필요)
- **@xterm/xterm** — 터미널 UI
- **monaco-editor** — 코드 에디터
