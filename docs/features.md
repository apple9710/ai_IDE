# 기능 가이드

Claude IDE의 사용자 기능을 정리한 문서입니다.

---

## 1. 프로젝트 탭 (여러 프로젝트 동시 작업)

상단 **프로젝트 탭 바**에서 여러 폴더를 각각의 탭으로 열어두고, 크롬 탭처럼 전환할 수 있습니다.

- **추가**: 프로젝트 탭 바의 `＋` 버튼, 또는 사이드바 헤더의 `열기` 버튼으로 폴더 선택
- **전환**: 탭 클릭, 또는 `Ctrl+Tab` / `Ctrl+Shift+Tab` 으로 순환
- **닫기**: 탭의 `✕`

### 프로젝트별로 유지되는 상태

탭을 전환해도 각 프로젝트는 아래를 **통째로 그대로 유지**합니다. 다른 프로젝트로 갔다가 돌아와도 하던 작업이 보존됩니다.

- 파일 트리(펼침 상태 포함)와 Git 상태
- 열려 있던 **에디터 탭들**
- **터미널** (백그라운드 프로젝트의 터미널도 계속 실행됨)
- **Claude 채팅 세션과 대화 내용** — 프로젝트마다 독립된 세션

> 마지막에 열려 있던 프로젝트 탭 구성과 활성 탭은 다음 실행 시 자동 복원됩니다.

---

## 2. 코드 에디터 (Monaco)

- 파일 트리에서 텍스트 파일을 클릭하면 에디터 탭으로 열립니다.
- 문법 강조는 확장자로 판별 (js/ts/py/go/rs/json/md/html/css 등).
- `Ctrl+S` 로 현재 파일 저장 (저장 후 Git 상태 자동 새로고침).
- 파일 탭의 `✕` 로 닫기.

---

## 3. PDF · 이미지 뷰어

텍스트가 아닌 파일도 에디터 영역에서 바로 볼 수 있습니다.

- **PDF** (`.pdf`): 내장 뷰어(iframe)로 렌더링
- **이미지** (`.png .jpg .jpeg .gif .webp .bmp .svg .ico`): 이미지로 표시

파일 트리에서 클릭하면 일반 파일 탭처럼 열리며, 텍스트 편집기가 아닌 뷰어로 표시됩니다. (읽기 전용)

---

## 4. 터미널

- 하단 패널의 실제 PTY 터미널 (일반 셸 — npm/git 등 자유롭게 사용).
- `＋` 로 여러 개 생성, 탭으로 전환.
- 패널 상단 경계를 드래그해 높이 조절.
- 터미널은 **현재 프로젝트 폴더**를 작업 디렉터리로 시작합니다.

---

## 5. Claude 채팅 패널 (Agent SDK)

우측 `✳ Claude` 패널은 `@anthropic-ai/claude-agent-sdk`로 Claude Code 엔진을 채팅형 UI로 구동합니다.

- 스트리밍 응답(토큰 단위), 툴 호출 카드(펼치면 입력/결과 확인), thinking 표시
- 권한이 필요한 작업은 **허용 / 항상 허용 / 거부** 카드로 승인 (이미 허용 규칙이 있으면 자동 허용)
- 대화는 열려 있는 프로젝트 폴더(`cwd`) 기준으로 동작
- `＋ 새 대화` 로 세션 초기화, `■` 로 진행 중 작업 중단
- **프로젝트마다 독립 세션** — 탭을 바꿔도 각 프로젝트의 대화가 유지됩니다.

인증은 설치된 **Claude Code CLI 로그인(`claude`)을 그대로 사용**합니다. 터미널에서 한 번 로그인해두면 별도 설정 없이 동작합니다.

### 5-1. 모델 · 추론 강도 선택

채팅 헤더 아래 설정 줄에서 선택합니다. 선택값은 저장되어 다음 실행에도 유지됩니다.

- **모델**: 기본값 / Opus 4.8 / Sonnet 5 / Haiku 4.5 / Fable 5
  - "기본값" 은 CLI 기본 모델을 사용
  - 모델 변경은 진행 중인 세션에도 실시간 적용됩니다.
- **추론(effort)**: 낮음 / 보통 / 높음(기본) / 매우 높음 / 최대
  - "최대" 는 일부 모델만 지원하며, 미지원 모델에서는 자동으로 한 단계 낮춰 적용됩니다.
  - 추론 강도 변경은 다음 새 대화(세션)부터 적용됩니다.

### 5-2. 이미지 첨부

메시지에 이미지를 함께 보낼 수 있습니다 (모델이 비전을 지원하는 경우).

- **추가 방법 3가지**: 입력창의 📎 버튼(파일 선택, 여러 장 가능) / **붙여넣기(Ctrl+V)** / **드래그 앤 드롭**
- 첨부하면 입력창 위에 썸네일로 표시되고, ✕ 로 개별 제거
- 전송하면 보낸 말풍선에도 이미지 썸네일이 함께 표시됩니다. (텍스트 없이 이미지만 전송 가능)
- 카카오톡 등에서 우클릭 > 복사한 이미지도 붙여넣기가 됩니다. (네이티브 클립보드 비트맵 폴백)
- **용량 제한**: 장당 5MB 초과 시 경고가 뜨고 첨부되지 않습니다.

### 5-3. 메시지 복사 · "이 부분 질문하기"

- 메시지에 마우스를 올리면 **"복사"** 버튼이 나타나 메시지 전체를 복사합니다.
- 채팅 텍스트를 드래그로 선택하면 위에 팝업이 뜹니다:
  - **복사** — 선택한 부분만 클립보드로 복사
  - **💬 이 부분 질문하기** — 선택한 텍스트를 인용(`> …`)으로 입력창에 넣고 포커스 (바로 후속 질문 작성)

## 6. Git 상태 표시

| 배지 | 의미 | 색상 |
|------|------|------|
| `M` | 수정됨 | 앰버 |
| `A` | 스테이징된 추가 | 그린 |
| `D` | 삭제됨 | 레드 |
| `U` | 미추적 파일 | 블루 |
| `R` | 이름 변경 | 블루 |
| `●` | (폴더) 내부에 변경 존재 | 앰버 |

- Git 저장소가 아닌 폴더를 열면 브랜치 바는 자동으로 숨겨집니다.
- 파일 저장·생성·삭제 후 자동 새로고침되며, 🔄 버튼으로 수동 새로고침도 가능합니다.

---

## 7. 파일/폴더 관리

- 사이드바 헤더 버튼 또는 트리 **우클릭 컨텍스트 메뉴**로 새 파일·새 폴더·이름 변경·삭제.
- 삭제는 확인 대화상자를 거칩니다.

---

| 키 | 동작 |
|----|------|
| `Ctrl+B` | 탐색기(사이드바) 토글 |

| `Ctrl+L` | Claude 채팅 패널 토글/포커스 |
| `Ctrl+S` | 현재 파일 저장 |
