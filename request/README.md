# request — IDE 개선 요청함

IDE를 쓰다가 막히는 것, 버그, 원하는 기능이 있으면 **이 폴더에 파일 하나를 만들어 적어두세요.**
백그라운드에서 Claude Code가 주기적으로 이 폴더를 확인해서 처리합니다.

## 사용법

1. 이 폴더에 `.md` 또는 `.txt` 파일을 만든다 (파일명 자유, 예: `terminal-resize.md`).
2. 내용은 자유 형식 — 증상/원하는 것만 적으면 됨. 스크린샷 설명, 재현 방법이 있으면 더 좋음.
3. 저장하면 끝. 처리가 끝나면 파일 하단에 **처리 결과**가 추가되고 `done/` 폴더로 이동됨.

## 규칙 (자동 처리기가 따르는 것)

- 요청 구현 후 앱 부팅 스모크 테스트를 통과해야 커밋함.
- CHANGELOG.md 갱신 + SemVer 버전 상승 후 커밋·푸시.
- 판단이 애매하거나 위험한 요청(대량 삭제, 외부 배포 등)은 처리하지 않고 파일에 질문을 남김.
- 반영된 변경은 IDE를 재시작해야 보임.
